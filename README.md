# Описание

Веб-приложение предоставляет функционал конвертации валют на основе последних валютных курсов, опубликованных ЦБ РФ.

Backend размещен на **heruku**.

Frontend размещен на **GitHub Pages**.

## Используемые технологии:

### Backend

- Spring Boot

- Liquibase

- GraphQL

### Frontend

- React

- MobX

- Apollo
